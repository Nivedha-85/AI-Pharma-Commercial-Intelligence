# Project Charter

## Project Name

AI-Powered Pharmaceutical Commercial Business Insights Platform

---

## Background

Commercial teams analyse prescription data to understand product performance, prescribing trends and regional opportunities.

Current reporting is largely manual and time consuming.

The objective of this project is to build a centralized analytics platform capable of providing interactive dashboards, commercial insights, demand forecasting and AI-assisted summaries.

---

## Business Problem

Commercial reporting is fragmented across multiple data sources.

Decision makers require faster access to reliable insights regarding:

- Prescription trends
- Drug performance
- Regional performance
- Provider behaviour
- Future demand

---

## Project Goal

Develop an end-to-end analytics platform using the CMS Medicare Part D dataset.

---

## Success Criteria

- Automated data ingestion
- Clean analytical dataset
- SQL data warehouse
- Executive Power BI dashboard
- Forecasting
- AI-generated business summaries
