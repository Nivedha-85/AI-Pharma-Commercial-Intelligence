# AI-Powered Pharmaceutical Commercial Intelligence Platform

An end-to-end data analytics platform that transforms raw CMS Medicare Part D prescription data into actionable commercial insights for pharmaceutical organizations.

The project demonstrates a complete analytics workflow, from automated data ingestion and data quality assessment to SQL analytics, interactive dashboards, and AI-powered business insights.

---

## Project Objectives

- Automate ingestion of CMS Medicare Part D prescription data
- Perform data profiling and quality assessment
- Clean and validate data for analytical use
- Design a PostgreSQL data warehouse
- Develop business-focused SQL analytics
- Build an executive Power BI dashboard
- Generate AI-powered commercial insights from analytical results
